AMR with LiDAR for Dynamic Obstacle Avoidance

ROS2 2D LiDAR SLAM-Toolbox AMCL Differential-Drive

Project Overview

Designed and deployed a ROS2-based SLAM and navigation stack on a differential-drive Autonomous Mobile Robot (AMR). By utilizing a 2D LiDAR sensor, the robot achieves real-time mapping, accurate localization, and dynamic obstacle avoidance in complex indoor environments.

Key Implementation Details

  • Real-Time Map Generation: Utilized SLAM-Toolbox to process sensor data and generate high-fidelity occupancy grid maps of unknown indoor environments.
  • Robust Localization: Integrated AMCL (Adaptive Monte Carlo Localization) on the generated maps to ensure high-precision positional tracking. This significantly improved navigation stability in dynamic and changing settings.
  • Dynamic Navigation: Configured the ROS2 navigation stack to continuously process LiDAR data, enabling the AMR to dynamically recalculate trajectories and avoid moving obstacles in real-time.
