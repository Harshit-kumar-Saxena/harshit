Crack Detection Bot in Pipes

YOLOv4-Tiny Computer Vision Embedded Edge AI CAD / SolidWorks Mechatronics
🏅 Patent Granted:
Successfully filed a patent application for the modular mechanical design and adaptive inspection methodology of this robot.

Project Overview

Developed an intelligent, in-pipe inspection robot designed to navigate through variable pipe diameters. By fusing a custom-engineered mechanical structure with a high-speed computer vision model deployed on embedded hardware, the bot autonomously identifies internal structural defects and cracks in real-time.

Key Implementation Details

  • Real-Time Perception: Integrated a camera-based system and trained a custom YOLOv4-Tiny model to detect cracks. The model was highly optimized to achieve real-time inference speeds (up to 30 FPS) directly on the embedded hardware.
  • Adaptive Hardware Design: Engineered a complex mechanical 6-bar slider system that allows the robot to seamlessly expand or contract to fit varying pipe diameters during active inspection.
  • Precision Localization: Implemented an encoder and servo-based localization system to accurately track the robot's position within the pipe, mapping exactly where structural defects are located.
