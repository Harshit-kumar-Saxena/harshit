Vision-Language-Action Robotic Arm

ROS2 MoveIt2 LLaMA 3B Ollama 5-DOF Kinematics
🏆 Vishwakarma Awards Semi-Finalist: Secured a Top 40 rank out of 1012 competing teams nationwide for this innovative approach to collaborative robotics.

Project Overview

Engineered a cutting-edge collaborative robotic arm that bridges the gap between natural human language and physical robotic execution. By integrating a local Large Language Model (LLM) directly into a ROS2 manipulation pipeline, the system can interpret complex spoken commands and translate them into precise pick-and-place operations.

Key Implementation Details

  • Manipulation Pipeline: Built a robust ROS2 and MoveIt2 software stack tailored for a custom 5-DOF robotic arm, enabling collision-free grasp planning and smooth kinematic execution.
  • VLA Integration: Embedded a Vision-Language-Action (VLA) framework using a local LLaMA 3B model via Ollama. This allows the system to process contextual natural language instructions without relying on external cloud APIs.
  • Goal Generation: Architected the translation layer that converts the LLM's semantic output into actionable motion goals and Cartesian coordinates for the MoveIt planning scene.
